**Explicación del Factorial:**

El factorial de un número, representado por la
operación "!", significa la multiplicación de
todos los números enteros positivos menores o
iguales al número en cuestión. 

**Cálculo del Factorial:**

Para calcular el factorial de un número N, sigue
estos pasos:

1. **Si N es igual a 0:** El factorial de 0 (0!)
es igual a 1.
2. **Si N es un número positivo:** Calcula el
factorial de N como: N! = N * (N - 1) * (N - 2) *
... * 2 * 1

**Ejemplo:**

* **Calcular el factorial de 5 (5!):**  5! = 5 * 4
* 3 * 2 * 1 = 120


**Ejemplos con Python:**

En Python, puedes calcular el factorial de un
número usando la función `math.factorial()` :

```python
import math

numero = 5
factorial = math.factorial(numero)
print(f"El factorial de {numero} es: {factorial}")
```
